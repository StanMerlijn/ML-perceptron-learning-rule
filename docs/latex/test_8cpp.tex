This file contains a series of test cases to verify the functionality of the \doxylink{class_perceptron}{Perceptron} and Perceptron\+Layer classes. The tests include training and prediction for various logic gates.

Test Cases\+:
\begin{DoxyItemize}
\item \doxylink{class_perceptron}{Perceptron} for AND Gate\+: Tests the perceptron\textquotesingle{}s ability to learn the AND gate.
\item \doxylink{class_perceptron}{Perceptron} for XOR Gate\+: Tests the perceptron\textquotesingle{}s ability to learn the XOR gate.
\item \doxylink{class_perceptron}{Perceptron} for Iris Data Set\+: Tests the perceptron\textquotesingle{}s ability to learn the Setosa and Versicolor classes.
\item \doxylink{class_perceptron}{Perceptron} for Iris Data Set\+: Tests the perceptron\textquotesingle{}s ability to learn the Versicolor and Virginica classes.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The tests use the Catch2 framework for unit testing.

Test case for the XOR gate using the \doxylink{class_perceptron}{Perceptron} model. 

This test case trains a perceptron on the XOR gate truth table. Since the XOR function is non-\/linearly separable, a single perceptron cannot correctly learn the XOR behavior. Therefore, the expected behavior is\+:
\begin{DoxyItemize}
\item The perceptron incorrectly predicts the output for \{0, 0\} (i.\+e. output is not 0).
\item The perceptron correctly predicts the output for \{0, 1\} (i.\+e. output is 1).
\item The perceptron incorrectly predicts the output for \{1, 0\} (i.\+e. output is not 1).
\item The perceptron correctly predicts the output for \{1, 1\} (i.\+e. output is 0).
\end{DoxyItemize}

The test prints the final weights and computed loss after training.
